캘리포니아 주택 가격 데이터 특성 설명¶

이 데이터셋은 캘리포니아 주택 가격 예측을 위한 데이터셋 입니다.

* MedInc: Median Income. 각 지역(블록 그룹)의 중간 소득
* HouseAge: House Age. 각 지역의 주택의 중간 연령
* AveRooms: Average Rooms. 각 주택당 평균 방 개수
* AveBedrms: Average Bedrooms. 각 주택당 평균 침실 개수
* Population: Population. 각 지역의 인구 수
* AveOccup: Average Occupancy. 각 주택당 평균 거주자 수
* Latitude: Latitude. 각 지역의 위도 정보
* Longitude: Longitude. 각 지역의 경도 정보
* target: Target. 타겟 변수로, 각 지역의 주택 가격의 중간값

import pandas as pd
from sklearn.datasets import load_iris
from sklearn.cluster import KMeans
import matplotlib.pyplot as plt

# Iris 데이터 로드
iris = load_iris()
X = iris.data

# 군집계산을 위한 빈 리스트 
cluster = []

# 군집 수를 1에서 10까지 변화시키면서 KMeans 알고리즘을 적용
for i in range(1, 11):
    kmeans = KMeans(n_clusters=i, init='k-means++', max_iter=300, n_init=10, random_state=42)
    kmeans.fit(X)
    cluster.append(kmeans.inertia_)  # 값을 리스트에 추가

# 군집 값을 그래프로 시각화하여 가장 꺾이는 점 찾기
plt.figure(figsize=(10, 6))
plt.plot(range(1, 11), cluster, marker='o', linestyle='--')
plt.title('iris Cluster_KMeans')
plt.xlabel('Number of Clusters')
plt.ylabel('cluster')
plt.show()
